# MCLan

Мод для подключения к LAN-миру Minecraft через WebRTC, без белого IP.

## Что в репозитории

- `mods/forge-1.19.2/mclantunnel-0.1.0-forge-1.19.2.jar`
- `mods/forge-1.20.1/mclantunnel-0.1.0-forge-1.20.1.jar`

Ставьте мод строго под вашу версию Minecraft/Forge.

## Как это работает (по-человечески)

1. Хост открывает мир в LAN.
2. Мод поднимает туннель и дает `Room code` (6 цифр).
3. Клиент вводит этот `Room code`.
4. У клиента появляется локальный адрес вида `127.0.0.1:PORT`.
5. Клиент подключается в Minecraft к этому адресу.

Белый IP не нужен.  
Для стыковки используется signaling-сервер `https://ppng.io` и STUN Google.

## Установка

1. Выберите jar под вашу версию игры.
2. Закиньте jar в папку `mods` у хоста и у клиента.
3. Версия Minecraft + Forge у обоих должна совпадать.
4. Набор модов (особенно контентных) у обоих должен быть совместим.

## Сценарий для хоста

1. Запусти Minecraft, зайди в мир.
2. `Esc -> Open to LAN -> Start LAN World`.
3. Открой экран `LAN Tunnel`.
4. Нажми `Detect LAN port`.
5. Нажми `Start HOST tunnel`.
6. Отправь другу `Room code`.
7. Дождись статуса, что клиент подключился.

## Сценарий для клиента

1. Запусти Minecraft с тем же модпаком.
2. Открой экран `LAN Tunnel`.
3. Введи `Room code` от хоста.
4. Нажми `Start JOIN tunnel`.
5. Посмотри адрес в статусе: `127.0.0.1:PORT`.
6. В Minecraft зайди в `Multiplayer` и подключись к этому адресу.

## Частые проблемы

### 1) `connection refused`

Обычно хост не открыл мир в LAN или LAN-порт изменился после перезапуска мира.  
Решение: на хосте заново `Open to LAN` и снова `Start HOST tunnel`.

### 2) `listen tcp :25565 ... bind ... address already in use`

У клиента порт 25565 уже занят другим процессом/старым туннелем.  
Решение: в `Client bind port` поставить другой порт (например `25566`), либо закрыть старый процесс туннеля.

### 3) Долгое `connecting` / `timeout`

Сеть может резать часть p2p-маршрутов.  
Решение: перезапустить туннель у обоих, проверить правильность `Room code`, попробовать еще раз.

## Важно

- Окно туннеля должно быть запущено во время игры.
- Если хост полностью перезапустил Minecraft/мир, обычно нужно заново запускать HOST-туннель и отправлять новый `Room code`.
